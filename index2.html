    How to setup inkscape for use with cutting machines (Work In Progress)

    This guide assumes you are using the latest version of Inkscape (1.0.1) at the time this was written

    When you use inkscape to make designs for cutting machines in inkscape the svgs you make will usually be much larger when you import them into the software you use to cut them. For example if you create an svg in that is just a 1 inch by 1 inch rectangle when you upload it to cricut design space and insert it into your project it will be 1.333 by 1.333 inches. This can really be really annoying when you're trying to fit a new cut or drawing onto an existing design, when you're making somehting like a card where you need to know the finished size or anything that requires precise measurements like my motion activated cards. I've found a few different ways to fix this issue, each with their own pros and cons that I'll cover here

    Scale everything in the design:

    This is the quickest and easiest method and if you just have a single object or a simple design or you mainly use other programs and only use inkscape ocassionally. It's much less complex than the others.

    1. In the top inkscape menu click on edit > select all in all layers
    2. Next click object from the top menu and click Transform
    3. In the dialog that comes up on the right side make sure scale proportionally is checked. Set the units to % and change one of the boxes to 75%. press enter/return and the other box will change to 75% also. Click Apply, then save the file.

    Add a custom unit to inkscape and make a template file:

    This is my preferred method, you don't have to make changes to the viewport here which is beneficial if you use inkscape for anything else.

    1. Find and open the units.xml file. On a windows system the location of this file depends on how you installed it. If you used an installer like the exe or msi files This file will be in C:\Program Files\Inkscape or C:\Program Files (x86)\inkscape. If you used the portable package, open the folder that you unzipped the package to after you downloaded it. The rest of the process is the same for each version. Open the share folder, then open inkscape, from there open the ui folder and open the file units.xml
    2. Create a new unit. In this tutorial I'll be copying the inch unit but you can apply this to any unit you want to use.
